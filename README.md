# Joke Redux
Joke Redux is a fun and dynamic web application built using **HTML5, Tailwind CSS, and React (with Redux)**. The app fetches jokes from an API and allows users to browse, favorite, and manage them with a clean, responsive interface.

# Features
- **Fetch & Display Jokes** – Load random jokes from an external API
- **State Management** – Uses Redux to manage global state robustly
- **Responsive Design** – Works beautifully across devices
- **Minimal & Modern UI** – Styled with Tailwind CSS for elegance
- **Smooth Interactions** – React + Redux power instant and predictable updates

# Technologies Used
- **HTML5** – for semantic structure
- **Tailwind CSS** – for responsive, utility-first styling
- **React** – for component-based UI and rendering
- **Redux** – for state management across the application
